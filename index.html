CSS Selectors and Specificity Cheat Sheet

Selector Type Syntax Description Specificity Value (A, B, C)
Type Selector element Selects all elements with the given tag name. 0, 0, 1
Class Selector .class-name Selects all elements with a matching class attribute. 0, 1, 0
ID Selector #id-name Selects the single element with a matching ID attribute. 1, 0, 0
Descendant Combinator A B Selects element B that is a descendant of element A. (Sum of A's and B's)
